Power BI Project


Zomato Sales Analytics Dashboard

The Zomato Sales Analytics Dashboard is designed to provide Zomato and its restaurant partners with actionable insights into sales performance, order trends, user behavior, and city-level dynamics. By integrating SQL-based sales data, this dashboard empowers data-driven decision-making, enabling businesses to optimize pricing strategies, marketing campaigns, and overall operational efficiency. Key features include an overview of orders and sales city-wise, detailed user performance metrics, and a comprehensive analysis of city performance, allowing for targeted interventions and maximized revenue growth.

Python Project

Bankruptcy Prediction Model using Machine Learning

Developed a machine learning model to predict the risk of bankruptcy using financial and operational data. Key steps included data cleaning, visualization, and model building with algorithms such as Logistic Regression, Decision Tree, Random Forest, SVM, and KNN classifiers. Achieved high accuracy, particularly with the Random Forest Classifier, which attained an accuracy score of 99%.

Car Price Prediction Model using Machine Learning

Objective: Developed a machine learning model to predict car prices based on various features.
Tools & Libraries: Python, Scikit-learn, Pandas, Matplotlib, Seaborn.
Results: Achieved an R-squared value of 81.5% on the training set and 78.7% on the testing set.
Enhancements: Improved model accuracy using ensemble methods like Bagging, Random Forest, and XGBoost Regressor.

Electric Vehicle Market Segmentation using Clustering

Identified 5 distinct customer segments for targeted marketing campaigns, allowing for personalized outreach and resource allocation. Provided data-driven insights, including a 1677% increase in EV registrations between 2017 and 2023 and a 0.9 correlation between electric and hybrid vehicle adoptions. Visualizations and analysis presented to stakeholders, including time-series graphs, correlation matrices, and cluster heatmaps, informed strategic decision-making for the startup's launch in India.

Sales Store Analysis

Sales analysis is the process of examining and evaluating sales data to gain insights into the performance of a company's products or services. It involves analyzing various aspects of sales, such as trends, patterns, customer behavior, and performance metrics, to make informed business decisions. Sales analysis helps companies understand their sales performance, identify areas of strength and weakness, pinpoint opportunities for growth, and develop strategies to improve sales effectiveness.
